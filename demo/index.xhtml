hotlink.js demo

hotlink.js makes it possible to hide referrer information for hotlinked images.

Without hotlink.js

With hotlink.js

index.xhtml:7img {
    border: 3px dashed red;
}

hotlink.js is a simple drop-in that won't require you to make any modifications to your CSS. This CSS works fine on both images.
